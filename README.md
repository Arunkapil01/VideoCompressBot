- يمكنك إنشاء مقاطع فيديو مضغوطة ولقطات شاشة أيضًا.
- يمكنك تعيين اسم الفيديو المخصص والصورة المصغرة.
- يتم حفظ سجل الحركة الى قناتك الخاصة.
- في الوقت الحالي ، قمت بتعيينه على 5 عمليات كحد أقصى في المرة الواحدة.
